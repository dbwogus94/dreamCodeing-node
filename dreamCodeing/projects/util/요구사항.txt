# 사진이 저장된 폴더 정리 프로그램 시나리오
 
1. Pictures에는 달별로 폴더를 만들어서 정리한다.
2. 폴더안에는 아이폰, 안드로이드로 찍은 모든 동영상과 이미지 파일이 들어있다.
3. 파일들을 사진과 동영상으로 분류해야 한다.
4. 동영상 파일들은 video폴더로 넣는다.
5. 스크린샷(.png)과 수정한 파일(.aae)는 백업하지 않기 때문에 chaptured 폴더에 넣는다.
6. 아이폰에서 파일을 보정하면 파일이름 앞에 E가 붙는다.
  보정파일은 백업하지만 원본 파일은 지우지는 않고 백업하지도 않기 때문에 따로 보관한다.
	>>> 보정이 있는 원본 파일은 duplicated 폴더에 넣는다.

# 폴더 구조
- 최상위 Pictures
- 달별 폴더마다. video, captured, duplicated 생성
- video에는 동영상(.mp4, .mov)
- captured에는 이미지(.png, .aae)
- duplicated에는 보정 파일이 있는 원본 파일만 저장

# 추가 요구사항
- Pictures폴더안 있는 폴더명을 받아서 해당 폴더 정렬 
	>> node photo ${폴더명} 
- 파일 이동시 동작에 대한 간단한 로그 출력

# 필요한 기능
- 폴더 생성(fs.mkdir) + 폴더 있는지 확인(fs.exists)
- 파일을 리스트를 읽기(fs.readdir)
- 파일의 확장자 읽기(path.extname)
- 파일 위치 이동 (fs.rename)
- 파일이름에 E가 들어가는지 검색(new RegExp("^E"))
